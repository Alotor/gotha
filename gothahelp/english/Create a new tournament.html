System

Mac-Mahon

Mac-Mahon System's placement is based on Mac-Mahon Score (MMS).
Pairing is made between players with same MMS.
It is the most popular system and is recommended for nearly all kinds of tournaments.
In OpenGotha, by default, games in Mac-Mahon System may be played with handicap.

Swiss

Swiss System's placement is based on Number Of Wins (NBW)
Pairing is made between players with same NBW.
It can be considered as a Mac-Mahon system where all players start with 0 as common SMMS (Starting MMS).
In OpenGotha, by default, games in Swiss System are played without handicap.

Swiss with categories

This system is deprecated and, for this reason, the choice has been disabled in "Create a new tournament" dialog box.
However, it is still possible to choose it by the "Change Tournament system" dialog box in "General" tab of "Tournament parameters" frame.
This system divides players in rank-based categories. Pairing is made internally in each category.
